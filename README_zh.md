# PodChat

[🇺🇸 English README](./README.md)

## 🎙️ 项目描述

PodChat 是一个基于 Next.js 的播客交互应用。它不仅能把播客音频处理成文字稿、摘要、说话人身份和知识参考内容，还能把播客里的说话人变成可对话、可发声的 AI 角色。

这个项目最核心的体验是：用户不只是“听播客”，而是可以在播客处理完成后，直接和节目里的说话人交流，甚至进行多说话人的群聊互动。

## ✨ 项目亮点

- 👥 多说话人群聊：用户可以同时和播客中的多个说话人沟通，而不是只和单个 AI 助手对话
- 🔊 直接克隆说话人声音：每个 AI 说话人的声音都直接来自对应播客说话人的声音克隆，沉浸感更强
- 🎭 保留人物身份与语气：不同说话人会保留各自的角色、语气和上下文信息
- 📝 自动结构化播客内容：支持生成文字稿、章节、摘要和知识参考内容
- 🌍 支持双语阅读体验：可结合文字稿翻译边听边看
- ⚙️ 配置方式灵活：可在内置设置页中接入 ElevenLabs、Firecrawl 和兼容 OpenAI 的模型服务

## 🚀 本地部署

```bash
git clone https://github.com/fatC210/PodChat
cd PodChat
bun install
bun run build
bun run start
```

应用启动后，打开 `/settings` 页面并配置所需服务即可。

常见配置包括：

- ElevenLabs：用于转写、说话人声音克隆和语音交互
- 兼容 OpenAI 的大模型接口：用于摘要和聊天
- Firecrawl：用于补充网页知识参考内容
